# Careers Portal - Kids Den School

This is a full-stack Job Application Portal built with React (Frontend) and Express (Backend).

## Project Structure

- `client/`: React frontend powered by Vite.
- `server/`: Express backend with MongoDB.

## Getting Started

### Prerequisites

- Node.js
- MongoDB (Running locally or a remote URI in `server/.env`)

### Installation

Install dependencies for both client and server:

```bash
npm run install-all
```

### Running the Project

Run both client and server concurrently:

```bash
npm run dev
```

The frontend will be available at [http://localhost:5173](http://localhost:5173) and the backend at [http://localhost:5000](http://localhost:5000).

## Features

- Multi-role support (Applicant, Recruiter, Admin)
- Job discovery and application workflow
- Dashboard for managing postings and applications
